# Course Images

Add your screenshot images to this folder with these exact filenames:

| Filename | Description |
|----------|-------------|
| `cursor-ide-code.png` | Cursor IDE showing code editor view |
| `cursor-empty-state.png` | Cursor empty/welcome state with menu options |
| `cursor-chat-input.png` | The Composer/chat input area close-up |

| `deepflo-homepage.png` | DeepFlo.App homepage |

## Tips

- Use PNG or JPG format
- Recommended width: 1200px or less (will be responsive)
- Keep file sizes reasonable (under 500KB each if possible)
- Images will be displayed with rounded corners and a subtle border
