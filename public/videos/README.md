# Course Videos

Add your video files to this folder.

## Supported Formats
- MP4 (recommended)
- WebM

## Suggested Videos for Getting Started Chapter

| Filename | Description |
|----------|-------------|
| `cursor-overview.mp4` | Quick tour of the Cursor interface |
| `first-project.mp4` | Creating your first project demo |
| `deepflo-setup.mp4` | Setting up DeepFlo.App |

## Tips

- Keep videos short (under 2 minutes each)
- MP4 with H.264 codec works best for web
- Consider compressing videos to reduce file size
- Videos will autoplay muted with controls visible
